# USB Rubber Ducky => Android

## Shortcuts

+ System Shortcuts
  | Shortcuts | Output |
  |:---------:|:------:|
  | Win + / | List of Keyboard Shortcuts |
  | Alt + A | Apps list |
  | Win + Enter | Home |
  | Esc | Back |
  | Alt + Tab | Recents |
  | Alt + F4 | Close current app |
  | Win + N | Notifications |
  | Win + Shift + A | Microphone access on/off |
  | Win + L | Lock screen |
  | Alt + PRTSCN | Capture window |

+ Apps Shortcuts
  | Shortcuts | Output |
  |:---------:|:------:|
  | Win + B | Browser |
  | Win + K | Calender |
  | Win + E | Email |
  | Win + S | Messages |
  | Win + P | Music |
  | Win + M | Maps |
  | Win + R | My Files |
  | Win + Y | Smart View |
  | Win + Z | Setting |
